# Python Network 0

This project focuses on basic HTTP concepts and the use of curl in Bash scripting.

## Tasks covered:
- Retrieving response body size
- Displaying response body
- Sending DELETE requests
- Listing supported HTTP methods
- Sending custom headers
- Sending POST requests with parameters

## Concepts:
- HTTP methods (GET, POST, DELETE, OPTIONS)
- Request and response headers
- curl command usage
- Query parameters and request body

All scripts use curl to interact with web servers and follow minimal and clean Bash scripting practices.
