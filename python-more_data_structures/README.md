# Python More Data Structures

This project focuses on advanced data structures in Python, including lists, sets, dictionaries, and functional programming concepts.

## Topics Covered
- Lists and transformations
- Sets and operations
- Dictionaries
- Lambda and map

## Tasks
- Matrix square
- Search and replace
- Unique sum
- Set operations
- Dictionary operations
- Advanced tasks (average, roman, etc.)

## Key Takeaway
Learning how to manipulate data efficiently using Python structures.

## Summary
This project improved my understanding of Python data structures.
